# 白云洞登山局 PWA 版

## 项目信息
- **类型**: PWA（Progressive Web App）
- **技术栈**: React + Vite + TypeScript + Supabase
- **部署**: Vercel
- **项目负责人**: Jimmie Ting（非专业开发者）
- **前身**: 微信小程序版（已归档在 jimmieting/baiyundong）

## 开发约定
- **所有代码注释、commit message、沟通语言统一使用简体中文**
- 代码风格由 Claude 自主决定，保持一致即可
- 每次重要改动都要通过 git 提交，方便回溯

## 视觉风格：水墨意境
- **背景**: 宣纸色 #F5F0E8
- **主文字**: 墨黑 #2C2C2C（宋体/衬线体优先）

- **辅助**: 烟灰 #A0937D、淡墨线 #E8E0D0
- **整体气质**: 中国山水画 + 碑刻匾额感

- 起点：埠兴村登山口 (26.070797, 119.372559)
